# Test-File-maker
 Creates almost all files. They are automatically filled with random characters.
 
==================This is how it is used================

1. Open .exe file
2. Enter a file path (example: C:\Users\\"username"\Desktop\test\test.txt
3. Make sure the file does not exist
4. Select an option
5. Wait until finish
==================Quick fix================

If it doesn't work, make sure your antivirus program isn't blocking the .exe file.
